Strawberry Profile

Warning: This command has been deprecated.
Recommended replacement: The 'config list' command

Usage

cmd-cli profiles list strawberry-profiles [options]

Options

  • --show-contents | --sc (boolean)

    • List strawberry profiles and their contents. All profile details will be printed as part of command output.

Examples

  • List profiles of type strawberry:

    • cmd-cli profiles list strawberry-profiles

  • List profiles of type strawberry and display their contents:

    • cmd-cli profiles list strawberry-profiles --sc

Kiwi Profile

Warning: This command has been deprecated.
Recommended replacement: The 'config list' command

Usage

cmd-cli profiles list kiwi-profiles [options]

Options

  • --show-contents | --sc (boolean)

    • List kiwi profiles and their contents. All profile details will be printed as part of command output.

Examples

  • List profiles of type kiwi:

    • cmd-cli profiles list kiwi-profiles

  • List profiles of type kiwi and display their contents:

    • cmd-cli profiles list kiwi-profiles --sc

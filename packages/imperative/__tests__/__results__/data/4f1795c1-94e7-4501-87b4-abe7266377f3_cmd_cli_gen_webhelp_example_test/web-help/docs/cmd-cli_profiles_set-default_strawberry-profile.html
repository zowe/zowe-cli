The strawberry set default-profiles command allows you to set the default profiles for this command group. When a strawberry command is issued and no profile override options are specified, the default profiles for the command group are automatically loaded for the command based on the commands profile requirements.

Warning: This command has been deprecated.
Recommended replacement: The 'config set' command

Usage

cmd-cli profiles set-default strawberry-profile <profileName> [options]

Positional Arguments

  • profileName (string)

    • Specify a
      profile for default usage within the strawberry group. When you issue commands within the strawberry group without a profile specified as part of the command, the default will be loaded instead.

Examples

  • Set the default profile for type strawberry to the profile named 'profilename':

    • cmd-cli profiles set-default strawberry-profile profilename
